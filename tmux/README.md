## Tmux configuration

`C` is control. `S` is Shift. Prefix is `C-a`. Cell size is 1 cursor.

*Shortcut:*

```
vertical split:                |

horizontal split:              _

move over panes:               H, J, K, L

move over the windows:         1, 2, 3, 4, ..

next window:                   ]

previous window:               [

resize (2 cells per bump):     S-Left|Right|Top|Bottom 

swap pane:                     { or }

select pane:                   o

reload page:                   r

```
